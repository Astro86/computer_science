## 웹의 구성요소
- 프록시 : 클라이언트와 서버 사이에 위치한 HTTP 중계자
- 캐시 : 많이 찾는 웹페이지를 클라이언트 가까이에 보관하는 HTTP 창고
- 게이트웨이 : 다른 애플리케이션과 연결된 특별합 웹 서버
- 터널 : 단순히 HTTP 통신을 전달하기만 하는 특별한 프록시
- 에이전트 : 자동화된 HTTP ㅇ요청을 만드는 준 지증적 웹클라이언트
